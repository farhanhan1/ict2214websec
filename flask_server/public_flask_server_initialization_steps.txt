Files (located at ~/myflaskapp):
requirements.txt
app.py
cookieDB.csv
cookie_classifier_model_RF.joblib (too big to put in repo, 250MB. generate through ml code)
distinctKeywords.txt (Needed for ML code to run)
features_StackingCode.py (ML code used to generate the joblib file)

Files (located at /etc/systemd/system):
myflaskapp.service

To run Flask server for API endpoint (after all files are present):
Ensure that port 5000 is open on both Azure Network Security Group and in VM Uncomplicated Firewall
sudo ufw allow 5000
sudo pip3 install -r requirements.txt
sudo systemctl daemon-reload
sudo systemctl restart myflaskapp.service
sudo systemctl status myflaskapp.service
